# Fuel Arena 프로젝트 개요

Fuel Arena는 주행 후 연비, 효율, 안정 주행, 동급 대비 성과를 점수로 바꾸고 그 점수로 다른 운전자와 경쟁하는 모바일 플랫폼이다.

핵심 가치는 “기름값을 아끼는 앱이 아니라, 이기고 싶어서 자연스럽게 아끼게 만드는 앱”이다.

## 타겟 사용자
- 출퇴근 주행을 반복하며 연비와 운전 습관을 개선하고 싶은 운전자
- 내 차량과 티어, 배지, 랭킹을 자랑하고 싶은 사용자
- 친구나 모르는 사람과 안전하고 비금전적인 경쟁을 즐기고 싶은 사용자

## 핵심 루프
가입 → 동의 → 차량 등록 → 주행 기록 → 점수 계산 → 랭킹 반영 → 배틀/미션 참여 → 보상 수령 → 프로필 성장

## 수익화 구조
- 선택형 리워드 광고
- 프리미엄 구독
- 시즌패스 상품
- 스폰서 챌린지와 쿠폰 제휴
