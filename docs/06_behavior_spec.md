# Behavior Spec

## 주행 기록
주행 시작 시 대표 차량과 오늘의 미션을 확인한다. 안전 모드에 들어가면 주행 시간과 거리만 표시하고 광고와 알림을 차단한다.

## 점수 계산
효율 점수, 안정 점수, 거리 보너스, 일관성 보너스에서 급가속, 급제동, 공회전 패널티를 제외한다. 비현실적 기록은 pending_review 상태가 된다.

## 랭킹 반영
verified score만 rankings에 반영한다. 공개 화면에는 닉네임, 티어, 점수, 차급, 연료 타입만 표시한다.

## 배틀 정산
배틀은 현금성 보상을 사용하지 않는다. 시즌 XP, 배지 조각, 쿠폰 응모권 같은 비금전 보상만 지급한다.

## 광고 보상
사용자가 선택할 때만 리워드 광고를 실행한다. 광고를 보지 않아도 기본 보상은 유지된다.

## 프리미엄 접근
프리미엄은 mock purchase 또는 in_app_purchase 상품 ID로 활성화한다. 광고 제거, 고급 통계, 라이벌 분석, 시즌패스 추가 보상이 열린다.

## 미션 진행
주행, 배틀, 광고 보상, 스폰서 챌린지 참여가 mission_progress를 증가시킨다.
