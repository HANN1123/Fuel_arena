# 차량 및 파워트레인 데이터 신뢰 등급 정책 (docs/31)

본 문서는 Fuel Arena 플랫폼에서 사용되는 차량 카탈로그 및 파워트레인 데이터의 신뢰성을 보장하고, 공정한 연비/효율 경쟁 환경을 유지하기 위한 절대적인 데이터 정책을 수립합니다.

---

## 1. 데이터 신뢰 등급 (source_status)

차량 카탈로그의 모든 파워트레인 변종(`vehicle_powertrains`)은 다음 등급 중 하나를 반드시 부여받아야 합니다.

1. **`verified_official` (공식 검증)**
   - 정부 공인 연비 데이터(예: 한국에너지공단) 또는 제조사가 공식 배포한 가격표/제원표와 완전히 일치하는 최고 신뢰도의 데이터.
2. **`verified_admin` (관리자 검증)**
   - 플랫폼 운영진이 공식 출처(공식 매뉴얼, 언론 보도 제원 등)를 직접 교차 검증하고 승인한 데이터.
3. **`imported_public` (공공 수입)**
   - 공공 데이터를 통해 대량 일괄 등록(Import) 완료되었으나, 세부 트림이나 변속기 등의 세부 스펙이 완전하게 교차 검증되지 않은 데이터.
4. **`user_submitted` (사용자 등록)**
   - 사용자가 카탈로그에 없는 차량을 직접 입력(Custom)하여 생성한 차량 데이터.
5. **`pending_review` (운영자 검토 중)**
   - 사용자가 직접 입력했거나, 데이터 충돌이 발생하여 관리자의 최종 승인/매핑을 기다리는 임시 데이터.
6. **`conflict` (출처 충돌)**
   - 공공 데이터와 제조사 제원, 혹은 사용자 입력 간 제원(배기량, 배터리 용량 등)이 서로 달라 검토가 필요한 상태.
7. **`deprecated` (사용 중단)**
   - 제원 오류나 판매 중단 등으로 인해 카탈로그 선택 목록에서 제외되거나 더 이상 신뢰할 수 없는 데이터.
8. **`unknown` (정보 부족)**
   - 연료 타입이나 리그 배정에 필요한 최소한의 제원이 누락되어 경쟁에 바로 참여할 수 없는 정보 부족 데이터.

---

## 2. 데이터 관리의 절대 원칙

1. **공식 경쟁 제한**:
   - `verified_official` 또는 `verified_admin` 등급을 받지 못한 데이터는 **공식 랭킹(시즌 리그 랭킹) 산정에서 전면 제외**합니다.
   - `pending_review` 및 `user_submitted` 등급의 차량은 오직 **개인 주행 기록 보관 및 친선 1:1 배틀**만 참여할 수 있습니다.
2. **충돌(Conflict) 데이터 격리**:
   - `conflict` 상태의 데이터는 신규 차량 등록 단계에서 선택 시 경고 뱃지를 표시하며, 기존 유저의 랭킹 등재가 보류됩니다. 관리자가 최종 병합 및 해결하기 전까지 공식 데이터로 승격되지 않습니다.
3. **공식 효율 정보 준비 중**:
   - 공인연비(`official_efficiency_combined` 등)가 불확실하거나 누락된 경우, 임의의 추정값을 강제로 채우지 말고 **`null`로 저장**해야 합니다.
   - UI에는 수치가 빈칸으로 나오는 대신 **"공식 효율 정보 준비 중"**이라는 일관된 안내 문구를 표시합니다.
4. **출처 명시 의무**:
   - `source_url` 또는 `source_name`이 채워져 있지 않은 모든 데이터는 절대로 `verified_official` 등급을 부여받을 수 없으며, CI 밸리데이터가 이를 감지하면 즉시 실패 처리합니다.
5. **연료 타입과 효율 단위 규격 통제**:
   - 전기차(`electric`)는 오직 `km/kWh` 단위만 사용해야 하며, 내연기관/LPG/하이브리드는 `km/L`, 수소차(`hydrogen`)는 `km/kg` 단위를 사용해야 합니다. 연료 타입과 연비 단위가 충돌할 경우 데이터 마이그레이션 및 저장 과정에서 검증 실패 처리합니다.

## 3. 출처(Sources) 우선순위

데이터 충돌 발생 또는 신규 차량 정보 등록 시 다음의 우선순위에 따라 데이터를 채택합니다.

1. **1순위**: 대한민국 정부 부처 및 공공기관 공인 연비/효율 데이터 (예: 한국에너지공단 수송에너지 데이터)
2. **2순위**: 차량 제조사 공식 제원표 또는 공식 가격표/카탈로그 PDF 사양
3. **3순위**: 수입차 공식 제원 페이지 및 제조사 영문 공식 카탈로그
4. **4순위**: 운영진이 신뢰할 수 있는 매체를 통해 검수한 내부 수동 제원 CSV
5. **5순위**: 사용자가 차량 등록증 및 공식 사양을 기반으로 제출한 정보
6. **최하위 (배제)**: 개발 편의를 위한 임시 seed/추정값 (제품 UI에서 verified 상태로 표시 금지)
