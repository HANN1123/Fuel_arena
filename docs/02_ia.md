# IA

## 정보 구조
- 계정: 인증, 동의, 개인정보, 프리미엄
- 차량: 대표 차량, 차량 목록, 등록/수정
- 주행: 시작, 안전 모드, 결과, 분석, 기록
- 경쟁: 랭킹, 배틀, 라이벌, 크루
- 성장: 시즌, 미션, 시즌패스, 배지, 업적
- 보상: 리워드 광고, 쿠폰, 스폰서 챌린지
- 운영: 관리자 KPI, 사용자, 주행, 광고, 신고, 설정

## 주요 라우트
`/splash`, `/onboarding`, `/auth/login`, `/auth/signup`, `/consent`, `/permissions`, `/vehicle/register`, `/home`, `/drive/start`, `/drive/safety`, `/drive/result/:sessionId`, `/ranking`, `/battle`, `/season`, `/mission`, `/profile`, `/rewards`, `/premium`, `/fairness`, `/settings`, `/admin`.

## 플로우
신규 사용자는 splash → onboarding → signup → consent → permissions → vehicle/register → vehicle/complete → home 순서로 이동한다.

주행 플로우는 home → drive/start → drive/safety → drive/result/:sessionId → ranking 또는 battle로 이어진다.
