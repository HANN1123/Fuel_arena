## 플로우
신규 사용자는 splash → onboarding → Google login → consent → setup → home 순서로 이동한다. 차량 설정을 선택하면 setup/vehicle → vehicle/complete → home으로 이동하고, 건너뛰면 home에서 차량 미설정 상태를 확인한다.

주행 플로우는 home → drive/start → drive/safety → drive/result/:sessionId → ranking 또는 battle로 이어진다.

주행 기록은 drive/history에서 최근 세션과 검증 점수를 목록으로 보여주고, drive/analysis/:sessionId에서 거리, 시간, 평균 효율, 점수 분석, 검토 요청 CTA를 확인한다. 정확한 좌표와 상세 경로는 이 흐름에서도 표시하지 않는다.

랭킹, 배틀, 시즌, 주행 시작은 대표 차량이 없으면 차량 설정 CTA를 노출한다. ranking/detail은 내 대표 차량 기준 리그의 상위 랭커와 내 주변 순위를 보여준다. 라이벌은 공개 랭킹 데이터에서 내 순위와 바로 위 추월 목표를 계산한다. 대표 차량을 바꾸면 내 연료 리그와 차급도 함께 바뀐다.

`/profile/:userId`는 `public_rankings`에 반영된 공개 경쟁 정보만 사용한다. 닉네임, 티어, 점수, 차급, 연료 리그만 표시하고 이메일, 정확한 위치, 상세 주행 경로, raw drive_points는 노출하지 않는다.

배틀 상세와 결과는 `/battle/detail/:battleId`, `/battle/result/:battleId`의 ID로 단일 배틀을 조회한다. 누락되거나 접근할 수 없는 ID는 첫 배틀로 대체하지 않고 배틀 목록으로 복구하는 빈 상태를 표시한다.

점수 보류나 랭킹 미반영 이의제기는 drive/result 또는 fairness → support/review-request → support/ticket/:ticketId 흐름으로 이어진다.
