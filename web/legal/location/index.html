Fuel Arena 위치정보 이용 고지

Fuel Arena는 주행 거리, 속도, GPS 정확도, 모의 위치 신호를 검증하기 위해 위치정보를 사용합니다. 정확한 좌표와 원본 주행 포인트는 공개 화면에 노출하지 않습니다.

수집하는 위치정보

  • 주행 중 위치 좌표, 측정 시간, 정확도, 속도, 이동 거리 계산에 필요한 신호
  • 모의 위치 여부와 비정상 속도 판단 값
  • 공식 주행 검증에 필요한 세션 요약과 검증 상태

이용 목적

  • 주행 거리와 효율 점수 계산, 연료 리그 배정, 배틀 및 랭킹 정산
  • 모의 위치, 비정상 속도, 낮은 GPS 정확도 같은 부정 기록 탐지
  • 신고, 이의제기, 공정성 검토에 필요한 범위의 기록 확인

공개 제한

  • 공개 프로필, 랭킹, 라이벌 화면에는 이메일, 정확한 위치 좌표, 상세 경로, raw drive_points를 표시하지 않습니다.
  • 원본 주행 포인트는 private table로 보관하고, 공개 화면에는 요약 지표만 표시합니다.
  • 주행 중에는 광고, 팝업, 도전장, 불필요한 알림을 표시하지 않습니다.

권한 관리

위치 권한을 거부한 경우 앱 설정에서 권한을 다시 열 수 있습니다. 위치정보 이용 문의는 앱 내 고객지원 화면에서 접수합니다.
